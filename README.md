# Instituto Aurora — Psicologia & Terapia Online

Landing page premium desenvolvida para o projeto conceitual Instituto Aurora, focado em psicologia e terapia online com experiência visual sofisticada, acolhimento emocional e identidade premium.

---

## ✨ Visão do Projeto

O Instituto Aurora foi desenvolvido com foco em:

- experiência premium nível agência
- visual cinematográfico leve
- branding sofisticado
- UX emocional
- performance otimizada
- responsividade real
- acessibilidade
- SEO técnico
- estrutura profissional para portfólio freelancer

O projeto combina estética editorial, design minimalista premium e arquitetura front-end moderna.
